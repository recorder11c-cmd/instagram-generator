# 小さなお店のための5問アンケート設計テンプレート

公式LINE・Instagram・店頭QRで、お客様の声を短く集めて改善につなげる

## はじめに

お客様の声は大事。

でも、忙しいお店ほど、なかなか丁寧に聞けません。

毎日の営業、予約対応、接客、仕入れ、SNS更新。
その中で、アンケートまで作るのは簡単ではありません。

ただ、最初から長いアンケートを作る必要はありません。

まずは5問ほどで、短く、早く聞いてみる。

それだけでも、

- 説明の分かりにくさ
- 購入前の不安
- また利用したい理由
- 改善した方がよい点
- お客様が本当に見ているポイント

が見えてくることがあります。

この記事では、小規模店舗・地域サービス向けに、

- 5問アンケートの作り方
- 公式LINEやInstagramで送れる配信文
- 回答をどう読み取るか
- 改善提案にどう変えるか

をテンプレート形式でまとめます。

## この内容の対象者

このテンプレートは、以下のような方に向いています。

- 小規模店舗を運営している
- サロン、整体、カフェ、教室、地域サービスを運営している
- 公式LINEを作ったが、配信以外に活用できていない
- Instagramはあるが、お客様の本音を拾えていない
- 新メニューやサービス改善の反応を知りたい
- 大きな調査ではなく、まず小さく試したい

逆に、以下の目的には向いていません。

- 大量回答を集めたい
- ポイント目的のユーザーを大量に集めたい
- 統計的に厳密な市場調査をしたい
- すぐ売上が上がる魔法の施策を探している

目的は、あくまで「お客様の声を小さく集めて、改善につなげること」です。

## なぜ5問でよいのか

顧客理解というと、大がかりな調査を想像しがちです。

でも、小さなお店や地域サービスでは、最初から完璧な調査を作るよりも、

```text
短く聞く
早く集める
すぐ直す
```

方が実用的です。

たとえば、5問だけでも以下のことは確認できます。

- 案内文は伝わっているか
- 利用したいと思えるか
- 不安な点はどこか
- 良いと思われている点は何か
- 次に直すべき点は何か

大切なのは、回答数を大きく見せることではありません。

次に何を直すかが見えることです。

## 基本の5問テンプレート

まずは、以下の5問から始めるのがおすすめです。

```text
Q1. 今回の案内内容は分かりやすかったですか？

Q2. 実際に利用・参加してみたいと思いましたか？

Q3. 不安に感じた点や分かりにくかった点はありますか？

Q4. 良いと思った点はありますか？

Q5. 改善した方がよい点があれば教えてください。
```

ポイントは、点数だけでなく「理由」を聞くことです。

点数や選択肢は集計しやすいですが、改善のヒントは自由記述に出ることが多いです。

## 公式LINEで送る文面テンプレート

```text
いつもありがとうございます。

サービス改善のため、短いアンケートにご協力いただける方を募集しています。

所要時間は約1〜3分です。
回答は任意です。

いただいた内容は、今後の商品・サービス改善の参考にさせていただきます。

ご協力いただける方はこちらからお願いします。
【アンケートURL】
```

謝礼がある場合は、条件を先に書きます。

```text
回答完了後、条件を満たした方へ後日ポイントを付与します。
ポイント付与予定日: 【日付】
```

曖昧にすると不安につながるので、所要時間、任意回答、利用目的、謝礼条件は先に伝えます。

## Instagram DMで送る文面テンプレート

```text
こんにちは。
サービス改善のため、短いアンケートを実施しています。

1〜3分ほどで終わる内容です。
もしよければ、率直なご意見をいただけると嬉しいです。

回答はこちらです。
【アンケートURL】
```

DMでは長すぎる説明は読まれにくいので、短くします。

## 店頭QRで案内する文面テンプレート

```text
お客様の声をお聞かせください

サービス改善のため、1〜3分の短いアンケートを実施しています。
今後の商品・接客・メニュー改善の参考にさせていただきます。

QRコードから回答できます。
```

店頭では「何のために聞くのか」を一目で分かるようにします。

## 回答の読み取り方

回答が集まったら、まず以下の3つに分けます。

### 1. 良かった点

- 分かりやすい
- 使ってみたい
- 興味がある
- 価格がちょうどよい
- 所要時間が短くてよい

これは、今後の案内文やLPで強調できます。

### 2. 分かりにくかった点

- 何をするサービスか分からない
- 誰向けか分からない
- 料金や条件が分からない
- 特典の受け取り方が不安

これは、すぐ修正できる改善ポイントです。

### 3. 次に試すこと

- 冒頭説明を短くする
- 対象者を明記する
- 所要時間を先に書く
- よくある質問を追加する
- 送信タイミングを変える

回答を読む時は、批判として受け取るより「次に直す材料」として扱います。

## 改善提案への変換例

### 例1

```text
回答: 何のサービスか少し分かりにくかった
```

改善案:

```text
LPや配信文の冒頭に「誰向けのサービスか」を追加する。
```

### 例2

```text
回答: ポイントがいつ付くのか分からなかった
```

改善案:

```text
案内文に「ポイント付与予定日」と「条件」を先に書く。
```

### 例3

```text
回答: 興味はあるが、今すぐではない
```

改善案:

```text
今すぐ購入ではなく、次回案内や相談予約につなげる文面を用意する。
```

## 業種別の質問例

### カフェ・飲食店

```text
Q1. 新メニューの内容は分かりやすいですか？
Q2. 実際に注文してみたいと思いますか？
Q3. 価格についてどう感じますか？
Q4. 写真や説明で気になった点はありますか？
Q5. 追加してほしいメニューや改善点はありますか？
```

### 美容室・サロン

```text
Q1. メニュー説明は分かりやすいですか？
Q2. 予約前に不安な点はありますか？
Q3. 価格や所要時間は分かりやすいですか？
Q4. また利用したいと思う理由は何ですか？
Q5. 改善してほしい点があれば教えてください。
```

### 整体・治療院

```text
Q1. 施術内容の説明は分かりやすいですか？
Q2. 初回予約前に不安な点はありますか？
Q3. どんな症状や悩みで利用したいですか？
Q4. 信頼できると感じる情報は何ですか？
Q5. 追加してほしい説明や改善点はありますか？
```

### 教室・スクール

```text
Q1. レッスン内容は分かりやすいですか？
Q2. 参加前に不安な点はありますか？
Q3. 料金やスケジュールは分かりやすいですか？
Q4. どんな目的で参加したいですか？
Q5. もっと知りたい情報はありますか？
```
