# レコルダ 初回営業アクションキット

作成日: 2026-07-10

## 目的

「はじめての顧客理解レポート」を、身近な事業者1〜3社へ個別提案し、最初の有償受注を作る。

公開LP:

https://recorder-llc.com/?business=20260710

## 今日の営業ゴール

- 候補先を3〜10件書き出す
- そのうち1〜3件へ個別連絡する
- 30分ヒアリングの日程を1件取る

いきなり大量送信しない。最初は相手の顔が分かる事業者へ、個別に送る。

## 優先候補

優先度が高い順。

1. 公式LINEをすでに使っている店舗・事業者
2. 口コミ、リピート、新メニュー、顧客満足に悩みがありそうな事業者
3. 自分との関係性があり、30分相談をお願いしやすい相手
4. 美容室、整体、サロン、飲食店、学習塾、小規模EC、地域サービス

## 30分ヒアリング台本

### 0〜5分: 挨拶と目的説明

```text
今日は売り込みというより、まず現状を聞いて、アンケートやLINEで何ができそうか整理する時間にしたいです。
```

### 5〜20分: 質問

1. 今、お客様について一番知りたいことは何ですか？
2. 最近、売上・集客・リピートで気になっていることはありますか？
3. 公式LINE、Instagram、顧客リストなど、お客様へ連絡できる手段はありますか？
4. 新商品・新メニュー・改善したいサービスはありますか？
5. お客様から直接聞きにくいことはありますか？
6. レポートとして見たいのは、数字、自由記述、改善提案のどれですか？

### 20〜27分: 提案例

```text
今のお話だと、まずは5問くらいで「利用理由」「不満点」「再来店のきっかけ」「新メニューへの興味」「改善要望」を聞くのが良さそうです。

回答を集計して、自由記述を要約し、改善アクションを5つに整理する形なら、初回レポートとして実施できます。
```

### 27〜30分: 次の確認

```text
もし進める場合は、こちらで質問案と配信文を作ります。
初回は33,000円で、7〜10営業日ほどで簡易レポートまで納品できます。
一度、質問案を作ってお送りしますか？
```

## 初回提案の価格説明

```text
初回は事例づくりも兼ねて、33,000円で考えています。
内容は、30分ヒアリング、質問5問程度の設計、配信文作成、回答集計、自由記述要約、改善提案を含む簡易レポートです。
```

```text
進める場合は、こちらで質問5問程度と配信文を作成し、回答集計と改善提案までレポートにまとめます。

質問案を作成してお送りしてもよろしいでしょうか。
```

### 返信がない場合

```text
先日はありがとうございました。
急ぎではありませんが、顧客理解レポートの件、必要なタイミングがあればいつでも相談ください。

新メニューの反応確認や、公式LINEの活用テーマでも使えます。
```

## 受注したら決めること

- 調査目的
- 対象者
- 配信方法
- 回答期限
- 謝礼の有無
- レポート納品形式
- 匿名事例として掲載可能か
