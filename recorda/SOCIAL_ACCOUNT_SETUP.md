# レコルダ X・note公式アカウント設定案

作成日: 2026-07-11

## 方針

レコルダは「ポイ活アカウント」ではなく、「顧客理解・LINEアンケート・小さな改善」の発信アカウントとして運用する。

ポイントは前面に出さず、回答協力への謝礼として扱う。

## 推奨アカウント名

### X表示名

```text
レコルダ｜顧客の声を改善につなげる
```

長すぎる場合:

```text
レコルダ｜顧客理解とLINEアンケート
```

### note表示名

```text
レコルダ合同会社
```

または

```text
レコルダ｜顧客理解レポート
```

## ID候補

優先順。

1. `recorda_voice`
2. `recorda_report`
3. `recorder_voice`
4. `recorder_llc`
5. `recorda_research`
6. `recorda_line`

Xとnoteで近いIDにする。

## プロフィール文

### X用 短め

```text
LINEやアンケートでお客様の声を集め、改善提案までまとめる「顧客理解レポート」を提供しています。小規模店舗・地域サービス・公式LINE活用を支援。レコルダ合同会社。
```

### X用 さらに短め

```text
お客様の声をLINEで集め、改善提案まで。小規模店舗・地域サービス向けの顧客理解レポートを提供しています。レコルダ合同会社。
```

### note用

```text
レコルダ合同会社です。LINEやアンケートでお客様の声を集め、商品・サービス改善につなげる「顧客理解レポート」を提供しています。小規模店舗、地域サービス、公式LINE活用、AI・業務改善について発信します。
```

## URL

```text
https://recorder-llc.com/
```

## 固定ポスト案

```text
レコルダ合同会社は、LINEやアンケートでお客様の声を集め、改善提案までまとめる「顧客理解レポート」を提供しています。

小さなお店や地域サービスでも、5問ほどの短いアンケートで見えることがあります。

声を、次の改善につなげる。
https://recorder-llc.com/
```

## 初回X投稿

```text
レコルダで、短いアンケートを小さく試しました。

大きな調査ではなく、まずは初回テスト。

回答導線、ポイント付与、集計まで確認できました。

少人数でも、

・説明が伝わるか
・答えやすい所要時間
・特典条件の不安

は見えてきます。

小さく聞いて、早く直す。
```

2026年7月18日に初投稿済み。リンクは本文に入れず、プロフィールURLへ誘導する方針。
