# レコルダ 収益化アクションパック

作成日: 2026-07-18

## 今日の目的

「モニターを増やす」だけで終わらせず、企業向けに売れる最小商品へつなげる。

売る商品は、まず以下に絞る。

## 初回商品

### はじめての顧客理解レポート

- 初回価格: 33,000円
- 対象: 小規模店舗、サロン、整体、飲食、塾、地域サービス、公式LINEを持つ事業者
- 内容:
  - 30分ヒアリング
  - 5問程度の質問設計
  - 配信文作成
  - 回答集計
  - 自由記述の匿名要約
  - 改善提案5個
  - 2〜4ページの簡易レポート
- 納期: 7〜10営業日

## 実績の見せ方

LPでは「9件」を大きく見せず、以下の言い方にする。

```text
レコルダ自身の公式LINEで短いアンケートを試験実施し、回答・ポイント付与・集計まで一連の流れを確認しました。

少人数のテストでも、説明の伝わり方や参加しやすい条件を具体的に把握できます。
```

営業時に聞かれた場合だけ、正直に以下を伝える。

```text
初回テストでは9件の回答が集まり、450ポイント付与まで確認しました。
件数を大きく見せるより、まず小さく聞いて改善点を整理する実証として扱っています。
```

## 個別提案メッセージ

知人・既存接点へ送る短文。

```text
こんにちは。レコルダで、LINEや短いアンケートを使ってお客様の声を集め、改善点を整理するサービスを準備しています。

初回メニューとして「はじめての顧客理解レポート」を33,000円で作成予定です。

5問程度のアンケート設計、配信文、回答集計、自由記述の要約、改善提案までまとめます。

公式LINEや既存のお客様リストがある店舗・サービスと相性が良いです。
もし周りで「お客様の声を取りたい」「新メニューの反応を知りたい」方がいれば、一度テスト価格で相談できます。
```


## note記事 下書き

タイトル:

```text
小さなアンケートから始める、レコルダの顧客理解テスト
```

本文骨子:

1. レコルダで短いアンケートの初回テストを行った
2. 目的は件数を大きく見せることではなく、回答導線・ポイント付与・集計が回るか確認すること
3. 少人数でも見えること
   - 説明が伝わるか
   - 回答したいと思えるか
   - 特典条件に不安がないか
   - 何分なら参加しやすいか
4. 企業側にとっての価値
   - 大きな広告費をかける前に反応を見られる
   - 新メニューやサービス改善の判断材料になる
   - 自由記述から改善のヒントを拾える
5. 初回メニュー「はじめての顧客理解レポート」の案内

## 最初の営業対象

優先順位:

1. 公式LINEをすでに持っている知人事業者
2. 美容室・サロン・整体などリピートが重要な業種
3. 飲食店・カフェなど新メニューの反応を知りたい業種
4. 塾・教室など保護者や生徒の声を取りたい業種
5. 小規模EC・地域サービス
