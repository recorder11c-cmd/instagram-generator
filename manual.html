デプロイマニュアル

Instagram 投稿文ジェネレーター ─ クライアント向け公開手順

─ ✦ ─

📋 概要

このマニュアルは、制作アンケートで生成したZIPファイルをNetlifyに公開するための手順書です。
クライアントごとに専用URLが発行され、そのURLを共有するだけでツールが利用できます。

デプロイ方法は2種類あります
① 自動デプロイ（推奨）: アンケートページのボタン一つで完了。所要時間：約30秒
② 手動デプロイ: ZIPをダウンロードしてNetlifyにドロップ。所要時間：約5分

🔧 事前準備（初回のみ）

  • Netlifyアカウントを作成する（netlify.com ─ 無料）
  • Anthropic APIキーを取得する（platform.claude.com）
  • Netlify Personal Access Tokenを発行する（下記参照）
Netlify Personal Access Tokenの発行方法：
app.netlify.com → 右上アイコン → User settings → Applications → Personal access tokens → New access token
アンケートページの「自動デプロイ設定」に一度入力すれば次回から自動入力されます。

⚡ 方法①：自動デプロイ（推奨）

1

アンケートページを開く

制作アンケートのURLをブラウザで開き、「自動デプロイ設定」欄にNetlifyトークンとAPIキーを入力する。

2

スクリーンショットをアップロード

クライアントのInstagramプロフィール・投稿のスクリーンショットを複数枚アップロードし「Claudeで解析する」をクリック。

3

情報を確認・編集して生成

Claudeが自動抽出した店舗情報を確認・必要に応じて修正し、「専用ジェネレーターをダウンロード」をクリック。

4

「Netlifyに自動デプロイ」をクリック

Step 4の画面で「⚡ Netlifyに自動デプロイ」ボタンをクリック。約30秒でURLが発行される。

5

URLをクライアントに共有

表示されたURLをコピーしてクライアントに送る。これで完了。

📦 方法②：手動デプロイ

1

ZIPをダウンロード・解凍

Step 4の「ZIPをダウンロード」ボタンでファイルを取得。ダウンロードフォルダに保存されたZIPをダブルクリックして解凍する。

2

Netlifyに新しいサイトを作成

netlify.com → Projects → 画面内の点線枠に解凍したフォルダをドラッグ＆ドロップ。デプロイが完了するまで待つ（約10秒）。

3

Environment variablesを設定

サイトのダッシュボードで：
Project configuration → Environment variables → Add a variable → Add a single variable
Key: ANTHROPIC_API_KEY
Value: sk-ant-xxxxxxxx（APIキーを入力）

4

再デプロイ

Deploys → 点線枠に再度フォルダをドロップ（環境変数を反映させるため）。

5

URLを共有

ダッシュボード上部に表示される xxxxx.netlify.app のURLをクライアントに送る。

💰 コスト目安

項目金額備考
Netlifyホスティング無料Freeプランで十分
Anthropic API約$0.003〜0.005 / 生成1回$20で約4,000〜6,000回
アンケート解析（Claude）約$0.01〜0.02 / 1クライアント画像解析込み
1クライアントあたりの初期費用はほぼ0円。APIの使用量はAnthropicコンソールで随時確認できます。

🛠 よくあるエラーと対処法

エラー原因対処
APIエラー (400)画像サイズが大きすぎるスクショを減らすか、より小さい画像を使う
APIエラー (401)APIキーが無効platform.claudeでキーを再確認・再発行
Failed to fetchファイル直接開き（file://）localhostまたはNetlify経由で開く
Page not foundURLにファイル名がついているファイル名を省いたURLで開く
自動デプロイ失敗トークンの権限不足Netlifyトークンを再発行（全権限付与）

📁 ZIPファイルの構成

shop_generator.zip
├── index.html               ← お客様専用のジェネレーター本体
├── netlify/
│   └── functions/
│       └── proxy.js         ← AnthropicAPI呼び出しプロキシ
└── netlify.toml             ← Netlify設定ファイル
このフォルダをまるごとNetlifyにドロップするだけでOK。ファイルを個別に扱う必要はありません。
